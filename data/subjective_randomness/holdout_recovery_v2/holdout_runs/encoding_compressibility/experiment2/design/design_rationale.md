# Design rationale — encoding_compressibility / experiment 2

## Models being discriminated

Five competing cognitive models of subjective randomness, with their key features:

| Model | Features used | Key prediction |
|---|---|---|
| **prototype_similarity** | `imbalance`, `p_alts` | Non-monotonic: penalizes sequences too far from prototype (θ_alt ≈ 0.65); fully-alternating looks *less* random than moderately-alternating |
| **inner_loop_model** | `imbalance`, `p_alts` | Identical to prototype_similarity (same architecture) |
| **bayesian_diagnosticity** | `n`, `h`, `alts` | Penalizes highly-alternating sequences as diagnostic of an alternating Markov generator; prefers p_alts ≈ 0.5 (fair-coin base rate) |
| **rle_description_length** | `alts`, `n` | Strictly monotonic: (alts+1)/n; fully-alternating *always* looks more random — no ideal rate |
| **max_run_length** | `max_run_norm` | Only max run matters; shorter max run = more random |

## Key discriminating axis

The EIG scoring selected 20 stimuli, all of the form **THTHT vs. [moderate-alternation sequence]**. This pair cleanly operationalizes the central theoretical tension:

- **THTHT** (n=5): p_alts=1.0, max_run_norm=0.0, rle_score=1.0
- **Moderate-alternation comparisons** (n=6 or n=8): p_alts≈0.4–0.43, max_run_norm≈0.29–0.40, rle_score≈0.50

Model predictions for THTHT vs. TTHHHT (representative pair, EIG=0.188):

| Model | Preference | Reasoning |
|---|---|---|
| rle_description_length | THTHT (A) | rle_score 1.0 > 0.5 — more alternations per character |
| max_run_length | THTHT (A) | max_run_norm 0.0 < 0.40 — shorter longest run |
| prototype_similarity | TTHHHT (B) | \|1.0−0.65\|=0.35 > \|0.40−0.65\|=0.25 — TTHHHT closer to prototype θ_alt |
| bayesian_diagnosticity | TTHHHT (B) | THTHT too alternating (resembles alternating Markov generator); TTHHHT closer to fair-coin p_alts=0.5 |

This 2-vs-2 split means the observed responses will directly update posterior mass across the theoretical divide between *monotonic compressibility accounts* (rle, max_run) and *non-monotonic prototype/diagnosticity accounts* (prototype, bayes).

## Candidate pool and EIG range

- **200 candidate pairs** generated by enumerating all H/T sequences of length 4–8, scoring each pair for model disagreement using proxy model scores, and selecting the top 200 by disagreement strength.
- EIG was computed from prior-predictive p_left distributions (no MCMC fit needed).
- **EIG range of selected stimuli: 0.177–0.188** (bits of information per trial).
- All 20 selected stimuli use THTHT as sequence A, with 4 length-6 comparisons (EIG=0.188) and 16 length-8 comparisons (EIG=0.177). The length-6 pairs score slightly higher because the moderate-alternation sequences are more compact and the contrast in max_run_norm is larger.

## How the design discriminates

Each trial asks: does the participant choose the fully-alternating short sequence (THTHT) as more random, or the longer moderate-alternation sequence? 

- If participants consistently choose the longer moderate-alternation sequence → supports prototype_similarity / bayesian_diagnosticity (too-alternating looks structured)
- If participants consistently choose THTHT → supports rle_description_length / max_run_length (more alternations / shorter max run = more random)
- Mixed responses will spread posterior mass accordingly
